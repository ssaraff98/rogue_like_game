# rogue_like_game
Rogue-like character based game written in Java

## Run game
```
cd src
javac game/Rogue.java
java game.Rogue <name_of_dungeon_xmlfile>.xml
```

## Delete all .class files from src directory
**Linux:** ```find src -type f -name "*.class" -delete```
**Windows:** ```del /s *.class```
